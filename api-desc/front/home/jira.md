# Jira - Home Feature

## Epic: Home Page & Navigation

### Story Points Estimate: 8

## User Stories

### US-001: View Home Page Sections
**As** a customer
**I want** to see curated home page content (sliders, categories, products, brands, coupons)
**So that** I can discover products and promotions

### US-002: Navigate Categories
**As** a customer
**I want** to browse the category tree in the navigation bar
**So that** I can find products by category

### US-003: Channel-Aware Content
**As** a customer
**I want** to see relevant products based on my browsing channel (Home vs Fast Shipping)
**So that** I get a tailored experience
