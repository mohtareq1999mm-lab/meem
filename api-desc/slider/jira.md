# Jira - Slider Feature

## Epic: Slider Management

### Story Points Estimate: 8

## User Stories

### US-001: CRUD Sliders
**As** an admin
**I want** to manage sliders with translatable titles and images
**So that** I can control promotional content on the home page

### US-002: Toggle Status
**As** an admin
**I want** to toggle slider active/inactive
**So that** I can control visibility

### US-003: Reorder
**As** an admin
**I want** to drag-and-drop reorder sliders
**So that** I can control display sequence
